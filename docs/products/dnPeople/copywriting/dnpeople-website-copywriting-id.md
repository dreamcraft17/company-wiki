## 7. FAQ SECTION

### Section Title
```
Pertanyaan yang Sering Ditanya
```

#### Q1: Apakah ada biaya tersembunyi?
```
Tidak. Harga yang Anda lihat di website adalah harga sebenarnya.
Tidak ada biaya setup, tidak ada "premium features" yang charge terpisah,
tidak ada surprise saat billing.

Hanya ada optional add-ons (misalnya white-label atau custom development),
dan itu dijelaskan di awal.
```

#### Q2: Bisa trial dulu sebelum bayar?
```
Ya! Ada dua cara:

1. Free tier: Gratis selamanya untuk sampai 50 karyawan.
   Full access ke semua fitur core (payroll, leave, attendance, talent dev).

2. Paid tier trial: 30 hari free di tier manapun, tanpa credit card.
   Coba Professional tier dengan full features.
```

#### Q3: Bisakah migrate dari Talenta?
```
Bisa! Kami punya dedicated migration service.
Kami handle:
- Export data dari Talenta
- Mapping ke dnPeople schema
- Training tim Anda
- Go-live support

Included di paket tahun pertama (no extra cost).
```

#### Q4: Support-nya bagus ga?
```
Tergantung tier:

FREE/STARTER: Email support (24 jam response)
PROFESSIONAL: Chat support (8 jam response)
BUSINESS: Priority support (4 jam response) + phone
ENTERPRISE: Dedicated account manager + 24/7 phone

Plus ada knowledge base lengkap, video tutorials, dan community forum.
```

#### Q5: Kapan ada mobile app native?
```
Roadmap: Q3 2027 (akhir tahun).

Sampai saat itu, dnPeople fully responsive di mobile browser.
Tidak sempurna seperti native app, tapi cukup untuk:
- Attendance check-in
- Leave request
- View payslip
- Lihat employee info

Saat mobile app launch, harga akan naik seiring fitur baru.
Existing customers keep current price untuk 12 bulan berikutnya.
```

#### Q6: Apakah cocok untuk perusahaan besar (1000+ karyawan)?
```
Cocok! Bahkan lebih cocok karena:
- Scalable architecture (DigitalOcean, global CDN)
- Multi-company support (holding + subsidiary bisa satu platform)
- Advanced customization (rule engine, conditional workflows)
- Volume discount (IDR 15-18K per employee untuk 1000+)

Pricing 1/8 dari SAP SuccessFactors atau Workday.
Fitur 80% sama.
```

#### Q7: Apakah support BPJS dan PPh 21?
```
Ya, fully support.

BPJS Kesehatan + Ketenagakerjaan: Auto-calculation sesuai nominal karyawan
PPh 21: Calculation + form download (1721 A1/A2) untuk lapor ke DJP
THR: Calculation + payment tracking
Tunjangan + Potongan: Custom components, unlimited jumlah
Differential tax treatment: Gross, net, gross-up methods

Update regulasi setiap kali ada perubahan (biasanya 1-2 minggu setelah DJP update).
```

#### Q8: Bagaimana kalau butuh fitur custom?
```
Ada tiga pilihan:

1. Feature request: Minta ke product team. Jika banyak customer yang butuh,
   kami prioritize di roadmap.

2. Custom development: Bayar IDR 100K-500K per feature untuk develop sesuai kebutuhan.
   Contoh: Custom salary formula, integration dengan ERP spesifik, etc.

3. Webhook + API: Developer Anda bisa build integrasi sendiri menggunakan API.
   Kami provide documentation lengkap.
```

#### Q9: Bagaimana dengan compliance (UU PDP, GDPR)?
```
dnPeople fully compliant:

UU PDP: Data karyawan stored locally (Indonesia). 
       Encryption at rest & in transit. 
       Audit log complete.

GDPR: Jika punya employees di EU, kami support data residency + consent management.

SOC2: Sedang dalam process (expected Q1 2027).

Lebih detail: Lihat Trust Center kami di website.
```

#### Q10: Bisa customize approval workflow?
```
Ya! Multi-level approval dengan conditional routing.

Contoh:
- Cuti tahunan <5 hari: Manager approve saja
- Cuti >5 hari: Manager + HR Director approve
- Advance payroll >IDR 50 juta: Manager + HR + Finance Director
- Izin kantor (WFH/business trip): Manager approve only

Tidak perlu coding, drag-drop UI untuk setup.
```
