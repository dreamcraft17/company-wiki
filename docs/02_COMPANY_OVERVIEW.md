## Tentang DN Tech

**DN Tech** (PT. Dozer Napitupulu Technology) adalah perusahaan teknologi yang berfokus pada pengembangan solusi digital untuk bisnis di Indonesia. Kami membantu perusahaan — dari startup/UMKM hingga enterprise — dalam transformasi digital melalui pengembangan software, produk SaaS internal, konsultasi IT, dan implementasi sistem.

## Layanan Utama

### 1. Enterprise Software Development

Pengembangan sistem bisnis custom: ERP, CRM, inventory management, workflow automation, dan HRIS untuk kebutuhan operasional perusahaan.

### 2. Web & Mobile Development

Website perusahaan, aplikasi mobile (iOS/Android), progressive web apps, dan platform digital customer-facing.

### 3. Cloud & DevOps

Migrasi cloud, CI/CD pipeline, containerization (Docker/Kubernetes), monitoring, dan infrastructure as code.

### 4. IT Consulting

Assessment teknologi, arsitektur sistem, digital transformation roadmap, dan technical due diligence.
